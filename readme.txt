Kips sync official extension allows you to synchronize products with the [Kips platform](https://kips.io/).

== Description ==
Please be careful ! [WooCommerce](https://fr.wordpress.org/plugins/woocommerce/) is required to make this plugin working.
Synchronize your [WooCommerce](https://fr.wordpress.org/plugins/woocommerce/) products catalog with the official [Kips platform](https://kips.io/) easily with this extension.

== Installation ==
1. Activate the plugin through the 'Plugins' menu in WordPress
2. Open the "Kips Options" page from the WordPress administration sidebar.
3. Fill in the Kips Client ID and Kips Client SECRET with your generated API keys and click on the Save bubtton.
